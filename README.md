# Street Rush — Endless Racer

An exciting retro top-down endless racing game built with HTML5 Canvas, JavaScript, and CSS. Dodge traffic, rack up points, and survive as long as you can! 

## Features
- **Endless Gameplay:** The road goes on forever, and the traffic gets faster!
- **Cross-Platform Support:** Fully playable on both desktop and mobile devices with touch controls.
- **Dynamic Graphics:** Smooth rendering that includes dynamic climate changes, simulating morning, sunrise, top-down lighting, and sunset effects as you drive further!
- **Simple Controls:** 
  - **Desktop:** Left/Right arrow keys or 'A' / 'D' to steer. Up arrow or 'W' to accelerate.
  - **Mobile:** Tap the left/right sides of the screen to steer.

## How to Run Locally
1. Clone the repository.
2. Open the `index.html` file in your favorite modern web browser.
3. Start driving!

## Technologies Used
- HTML5 Canvas
- Vanilla JavaScript
- CSS
